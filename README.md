# rn-toast

React Native cross-plateform (iOS/Android) toast notification component highly customizable.

## Installation

```sh
npm install rn-toast
```

## Usage

```js
import RnToast from "rn-toast";

// ...

const result = await RnToast.multiply(3, 7);
```
